# SPRINT 2 — Tourist Booking Flows

## Prompt for Replit:

Build all the tourist-facing booking pages. Load tour data from /data/tours.json, vehicles from /data/vehicles.json, transfers from /data/transfers.json, currencies from /data/currencies.json. Use the shared components built in Sprint 1.

### 1. Tours Listing (/app/tours)
- SectionHeading: overline "EXPLORE SRI LANKA", heading "Ready-to-go *tours*"
- **CYO banner at top:** Highlighted card with sage background: "Don't see what you're looking for?" + "Create your own tour →" button linking to /app/tours/custom
- **Filter bar:** Duration dropdown (Any, 5-7 days, 8-10 days, 11+ days), Region dropdown (All, Cultural Triangle, Hill Country, South Coast, Wildlife), Sort by (Popular, Price low-high, Price high-low, Duration)
- **Tour grid:** 2 columns desktop, 1 column mobile. TourCards for all 4 tours from tours.json
- Each card links to /app/tours/[slug]

### 2. Tour Detail (/app/tours/[slug])
Load tour by slug from tours.json. This is the MOST IMPORTANT page — make it beautiful.

- **Hero:** Full-width placeholder image area (gradient overlay), tour name (font-display, 40px), tagline, duration badge ("10 days / 9 nights"), difficulty badge
- **Overview section:** bg-white. Tour description paragraph. Highlights as pill chips (bg-sage text-forest-500).
- **What's included / not included:** Two columns side by side. Included: green check icon + text. Not included: red x icon + text. Light green/red tinted backgrounds.
- **Day-by-day itinerary:** bg-cream section. SectionHeading: "Your *itinerary*". Expandable accordion — each day card shows:
  - Day number circle (bg-forest-500 text-white, 36px round)
  - Title in font-display (e.g., "Colombo to Sigiriya")
  - Location badge
  - Description paragraph
  - Driving time badge (clock icon + "4–5 hours")
  - Key stops as small chips
  - Day 1 expanded by default, rest collapsed
- **MapPlaceholder** alongside itinerary on desktop (sticky), below on mobile. Shows all tour locations as waypoints.
- **STICKY BOOKING WIDGET** (right sidebar on desktop, fixed bottom sheet on mobile):
  - "Book this tour" heading (font-display)
  - Start date picker (DatePicker component, min date = today + tour's minLeadTimeDays)
  - "Add extra days" slider: -3 to +3, default 0. Label shows "10 days" updating to "11 days" etc.
  - VehicleSelector showing all 5 types with prices from this tour's vehicleRates
  - "Passengers" number input (1-35 based on selected vehicle capacity)
  - **Add-ons section:** Toggle switches for each add-on (mock: "Airport pickup £28", "Welcome pack £15", "Yala safari jeep £45"). Each toggle updates price.
  - **PriceSummary:** Shows: vehicle rate × days = subtotal, + add-ons, = total. Currency converted.
  - Cancellation note: "Free cancellation 10+ days before start" (green text, shield icon)
  - **CTA button:** "Book this tour — £650" (bg-amber-300 text-white rounded-full, large, full width). Price updates in real time.

### 3. Create Your Own (/app/tours/custom)
5-step wizard. Stepper component at top showing progress.

**Step 1 — Trip basics:**
- "What kind of trip?" — 4 selectable cards: Holiday, Business, Seasonal/Religious, Sport
- "How many travellers?" — number input
- "Preferred vehicle" — VehicleSelector component

**Step 2 — Destinations:**
- "Where do you want to go?" heading
- Grid of 12 destination cards (3 cols desktop, 2 mobile): Colombo, Sigiriya, Kandy, Ella, Galle, Yala, Trincomalee, Negombo, Nuwara Eliya, Tangalle, Anuradhapura, Haputale. Each card: placeholder image, name, short description. Selected = ring-2 ring-forest-500 with checkmark overlay.
- MapPlaceholder on the right showing selected destinations as dots
- "Anywhere else?" free text input at bottom

**Step 3 — Dates & duration:**
- Start date picker
- "How many days?" — slider from 3 to 21, with number displayed
- "Are your dates flexible?" — toggle switch

**Step 4 — Preferences:**
- Budget range: 4 radio cards (Budget, Mid-range, Premium, Flexible)
- Travel style: checkbox group (Adventure, Cultural, Relaxation, Mix)
- Interests: clickable tag chips (Nature, Wildlife, Beaches, Food & Cuisine, Temples & History, Tea Plantations, Surfing, Hiking, Photography). Selected = bg-forest-500 text-white.
- Special requests: textarea (500 chars, char counter)

**Step 5 — Review & submit:**
- Summary card showing all selections from steps 1-4
- Contact fields: name, email, phone, country dropdown (pre-fill if "logged in")
- Note: "Our team will review your request and send you a personalised quote within 24–48 hours."
- "Submit request" button (primary)
- On submit: show confirmation modal/page with reference number

Each step has "Back" and "Continue" navigation buttons. "Continue" validates required fields.
